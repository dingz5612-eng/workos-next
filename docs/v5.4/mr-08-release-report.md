## Go Evidence

- CheckoutStarted creates RoomInspection WorkItem.
- RoomInspected damage creates DepositSettlement WorkItem.
- RoomInspected cleaning creates ServiceTask WorkItem.
- ServiceTaskVerified creates ResourceReleaseRequest WorkItem.
- ResourceReleaseRequest is routed to ResourceInventory.
- CaseClosurePolicy prevents false close.
- Case close failure returns blocker, owner, and resolveAction.
- Blocker can create resolution WorkItem.
- Duplicate close is deduped by process run trigger/rule.
- Claim prevents concurrent handling through Claim/SLA Lite guard scope.
- Overdue WorkItems are visible in Manager Lite.
