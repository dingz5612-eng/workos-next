# Acceptance Rules Index

This directory defines release and Go/No-Go acceptance authority. It translates
engineering rules into evidence that can block or allow a release batch.

## Authority

- Rule precedence source: `docs/rules/v5.5/rule-authority.yml`.
- V5.4 release acceptance: `12-release-go-no-go.md`.
- V5.5 Rules OS acceptance: `13-v5.5-rules-os-go-no-go.md`.

## Acceptance Order

1. P0 WON-18 Gate Green evidence.
2. V5.4 Control Plane Guards evidence.
3. V5.5 rule authority and directory evidence.
4. API boundary, fact ownership, MR contract, invariant maturity, GateResult,
   and rule drift evidence.
5. Final V5.5 Go/No-Go.

If an acceptance rule is missing evidence, the result is `NO-GO` until the
evidence is generated by machine checks.
